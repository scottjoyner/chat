# Strategy Catalog (Expert)

The registry now contains legacy production strategies plus advanced strategies, totaling **50+**.

Each advanced strategy includes:

- purpose
- regime suitability
- required data
- required latency budget
- sizing model
- risk ceilings
- expected holding horizon
- execution style
- failure modes
- disable criteria
- cooldown logic
- explainability output
- backtest caveats
- live deployment prerequisites

See `strategies/catalog/advanced.py` for the complete machine-readable spec list.
